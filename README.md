# NodePad Documentation

Source for the NodePad docs site at **[docs.node-pad.com](https://docs.node-pad.com)**.

NodePad is a canvas workspace for AI chat. Learn more at [node-pad.com](https://node-pad.com) or [request beta access](https://node-pad.com/beta).

## Local preview

Install the [Mintlify CLI](https://www.npmjs.com/package/mint):

```bash
npm i -g mint
```

From the repo root (where `docs.json` lives):

```bash
mint dev
```

Preview at `http://localhost:3000`.
